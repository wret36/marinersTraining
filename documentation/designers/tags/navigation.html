pyro:navigation

The navigation tag creates lists of navigation links based on navigation groups defined in the Design -> Navigation section of PyroCMS.

Tags

{pyro:navigation:links}


Creates a list of links for a group.

Parameters

Parameter Default Description
group Required. The navigation group the tag should use.
tag li Type of tag that surrounds each navigation item. The value of this wrapped in brackets on either end.
class current The class to use when an element is the current page.
separator String that separates each navigation item.

Example

{pyro:navigation:links group="header" separator="|"}
<li>
	<a href="http://localhost/PyroCMS/pyrocms/index.php" class=" current">Home</a></li> |
	<li><a href="http://www.google.com">About Us</a>
</li>
