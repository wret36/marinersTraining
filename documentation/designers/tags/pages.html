pyro:pages

The pages tag displays page info.

Tags

{pyro:pages:url}


Gets a page's URL based on its ID.

Parameters

Parameter Default Description
id Required. Page ID the URL the tag should return.

{pyro:pages:children}


Tag pair that loops through the children of a parent page.

Parameters

Parameter Default Description
id Required. Page ID of the page you want the children for.
limit Required. Number of items to return.

Example

{pyro:pages:children id="1" limit="2"}
	<h2>{title}</h2>
	{body}
{/pyro:pages:children}
<h2>Child One</h2>
Body Content One
	
<h2>Child Two</h2>
Body Content Two
